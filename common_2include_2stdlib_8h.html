◆ malloc()

void* malloc ( size_t  size)

Simple malloc/free implementation.

Note: if your program does not need to use the heap, then do not call any of these functions. The entire heap implementation is only allocated in your program if you actually call the allocation functions.

Currently the heap is implemented using a first-fit free list. The heap segment is placed directly after BSS in your program. The heap can utilize any of the memory between the base heap address and the top of the software-defined stack.
